## What is NETCHAT?

NETCHAT is a TI-84 Plus CE program powered by TINET!  
With NETCHAT, you can chat with almost anyone!

## How to use

To install NETCHAT and setup TINET follow the [official TINET docs](https://tinetdocs.tkbstudios.com/)
